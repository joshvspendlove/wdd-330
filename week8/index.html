Week 8 Notes

  • When manipulating the content of a page, css is quicker and demands less from the processor.
  • You can even create animations with CSS using @keyframes.
  • The Canvas element can be used to draw content on a page. If it is not supported, whatever is listed between the two tags will be displayed to the user.
  • Canvas was first developed by Apple then adopted by other browsers.
  • Within the canvas tag, width and height attributes need to be defined. These determine the coordinate system for the canvas.
