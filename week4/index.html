Week 4 Notes

  • Form elements can be assigned to variables using the name attribute. For example: "const form = document.forms.search;"
  • Additional variables and functions can be added to an object's prototype after the object is created. All instances of the object will then inherit these variables and functions.
  • Variables declared in the constructor of an object become private variables. Getters and/or Setters should be created for these variables.
  • Monkey-patching is the process of adding your own functions to built-in JavaScript objects.
  • Here are two useful function that can be used to create an exact copy of an object.
					
    function mixin(target,...objects)
    {
      for (const object of objects)
      {
        if (typeof object === 'object')
        {	
          for (const key of Object.keys(object)) 
          {
            if (typeof object[key] === 'object')
            {
              target[key] = Array.isArray(object[key]) ? [] : {};
              mixin(target[key], object[key]);				
            }
            else 
            {
              Object.assign(target, object);
            }
          }
        }
      }
    }

    function copy(target)
    {
      const object = Object.create(Object.getPrototypeOf(target));
      mixin(object, target);
      return object;
    }
