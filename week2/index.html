Week 2 Notes

  • Keep HTML, JavaScript, and CSS code in seperate files. This allows for one to be easily changed without breaking the other.
  • While its by the preference of the developer, in most instances it is best to create a base that works across most browers and add enhanced features for browsers that support it.
  • There are 2 "Weak" data structures that deal with the reference of objects verses the value. These are Weak Sets and Weak Maps. These data structures were created to help eliminate memory leaks when the referenced object is deleted.
  • DRY - Don't Repeat Yourself
  • The function declaration, "function mean(...values){}" allows for multiple values to be passed as arguments into the function. These arguments can then be accessed through the values array that is created.
