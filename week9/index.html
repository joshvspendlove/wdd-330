Week 9 Notes

  • Global variables are declared without a const, let, or var. These are owned by the window object.
  • Each domain is only allowed 20 cookies at 4 KB each. Cookies and Local Storage differ as cookies are for the server and LocalStorage is for the client.
  • Data can be added to an HTML page through attributes using the “data-“ suffix. This data can then be pulled using element.dataset.*.
  • Workers are the key to multitasking. These are declared such as: const worker = new Worker('task.js');
  • Messages are sent between the main and worker with: worker.postMessage() and self.postMessage().
  • A message listener must be created to utilize this.
  • Workers can be terminated with: worker.terminate(); or self.close();
